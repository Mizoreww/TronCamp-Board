01 / 概览

力觉,是视觉看不见的那一半世界

robosuite 的 Wipe 是天然的力觉密集任务:机械臂要以受控接触力沿桌面 擦除污渍——压力不足擦不掉、压力过大触发超力惩罚。机械爪腕部内置六维力/力矩 传感器,整场比赛的力信号即由此而来。

全卷统一用 π0.5(~3B,FluxVLA 旗舰)、LoRA 微调、官方固定训练 seed—— 榜单上比的是力觉算法,不是谁的模型大。三道递进必做题,重心在 T3: T1/T2 是保底与科学对照,T3 力觉改进竞技场决定主榜名次。

T2/T3 统一在"桌高 ±1cm 内随机偏移 + 视觉遮挡"扰动条件下计分——偏移视觉几乎不可察觉, 纯视觉策略会戳桌/悬空,力反馈的因果作用直接决定成绩。

03 / 比赛规则

赛制与红线

赛制

10 天纯线上;每队 2×A100-80G,官方 LoRA 路径,必做 3 run 起(T1 无力 + T2 有力 + T3 ≥1 改进)。全卷单模型 π0.5、单任务 Wipe、单仿真栈,版本锁定 FluxVLA 指定 commit,赛中不跟上游。

评测条件

T1 干净条件(无偏移无遮挡);T2/T3 统一扰动条件(桌高 ±1cm 随机偏移 + 视觉遮挡)。每模型 dev 50 / final 100 个独立场景 seed episodes,全模型逐 episode 配对。dev seed 表(含 corrupter salt)随 starter 公开——本地分 = dev 榜分。

提交限额

每队每日(UTC+8)至多 2 次提交,超额拒绝。截止时刻提交通道关闭,当时的 dev 榜即最终成绩(榜单页有倒计时)。

红线(违反量化分作废)

① 官方训练 seed、use_contact_obs、两套 config 的非力列字段不得改(机械 diff 项);② 推理期只用白名单观测,特权量只准做训练标签;③ 自定义预处理只许写在 preprocess_obs() 内;④ T2 配对纪律:同 seed/同数据/同步数,diff 仅限力列;⑤ 负结果如实报告照样计分,造假单题零分起。

04 / 评分标准

不只看成功率

SR 是基础层,但单看 SR 测不出"力"的价值——一个"成功但暴力"的策略(超压硬怼也能擦完)SR 很好看、力行为很糟糕。所以采用三层指标,主指标尽量用连续量。

单题量化分 = 0.45·擦除率 + 0.30·力质量分 + 0.25·Δ （T2 用 Δ₁,T3 用 Δ₂;Δ 为逐 episode 配对差，scorer 强制输出 95% CI）
榜单总分 = 0.20·T1 + 0.30·T2 + 0.35·T3 ＋ 跨题统评 15%（开源规范与文档质量，人评）
① 任务层

擦除率 / SR

任务做没做成。T1 用它做通关判定;T2/T3 中优先用连续完成度(50 集二值 SR 噪声约 ±10pp,连续指标统计效力高得多)。

② 力行为层

力质量分

做成的方式对不对——恒压贴合还是暴力硬怼。接触力在专家分布目标带(P10–P90)内的时间占比、超压占比、峰值惩罚、力方差。只由主办方 scorer 从落盘力轨迹重算。

③ 鲁棒层

力觉增益 Δ

力是否真正起了因果作用。Δ₁ = 有力 − 无力配对差,Δ₂ = T3 − 自家 T2。±1cm 偏移视觉不可察觉,纯视觉策略读不到桌高变化,Δ 列直接挂在排行榜上。

05 / 提交方式

自动评测,选手自报数字无效

fork 官方题目库 → 训练 → 提交 checkpoint → 评测集群自动跑分上榜。本地用同一套 harness 与 dev 种子,跑分与 dev 榜一致。

Fork 官方 Starter

脚手架、提交模板、本地评测命令内置;以锁定 commit 为 base。

训练并核验

用官方 LoRA config 训练;保留可在线核验的 wandb run。

提交 checkpoint

CLI scripts/submit.py 或评测站网页上传入口(自动算 SHA256 回填);每日 ≤2 次。

ZMQ 隔离评测

选手 policy 进程(GPU)与主办方仿真 harness(CPU、私有种子)进程隔离,选手代码碰不到仿真内部状态。T3 用你 fork 的代码起服务,自定义模块全生效。

自动算分上榜

统一 score_force.py 算三层指标,逐 episode 日志落盘;Dev 榜实时,Final 榜截止冻结。

# 命令行提交(T2 示例) sha256sum <checkpoint> python scripts/submit.py --server http://<评测服务器>:8080 --token <队伍token> \ --track T2 --ckpt-url <URL或网页上传的file_url> --ckpt-sha256 <sha> \ --git-url <你的fork> --git-tag <tag>
